# Engineering Career Curriculum: Zero-to-Production

**Objective:** Transition from fragmented hobbyist to Production-Ready Full-Stack Engineer (TS/FastAPI/Linux).
**Focus:** Technical Indispensability, Security, and System Stability.

## 🏗️ The Curriculum Roadmap

### Phase 1: Foundations (Weeks 1–4)

- **Topics:** Python Internals, TS Type System, HTTP/Networking, SQL Normalization.
- **Goal:** Understand the "How" behind the code.

### Phase 2: Backend Mastery (Weeks 5–12)

- **Stack:** Python, FastAPI, SQLAlchemy 2.0, PostgreSQL.
- **Goal:** Build the "Sentinel" Secure API.

### Phase 3: Frontend Systems (Weeks 13–20)

- **Stack:** TypeScript, React, Tailwind, React Query.
- **Goal:** State management and type-safe UI components.

### Phase 4: Systems & Security (Weeks 21–28)

- **Topics:** Linux Kernel, Bash Automation, Docker, AWS Infrastructure.
- **Goal:** Build "LogScanner" CLI and deploy the full stack.

## 🔬 Mastery System

1. **The Friday Audit:** Weekly summary of mental models.
2. **Atomic Commits:** No "fixed stuff" commits.
3. **The Ledger:** Personal documentation of every bug encountered.
